# Level Editor (Hex Grid Map Generator)

## Архитектура
Редактор уровней построен на базе Unity Editor (Odin Inspector) и разделен на данные, логику генерации и UI.

### Основные компоненты:
1. **`MapGenerationConfig` (ScriptableObject)**
   - Хранит все настройки генерации: префабы (гексы, горы, деревья, камни), материалы биомов, параметры шума (Perlin Noise) для лесов и границ, настройки плотности и симметрии.
2. **`MapGenerator` (C# Class)**
   - Ядро генерации. Не зависит от `MonoBehaviour`. Принимает конфиг, размеры, seed и `Transform` родителя.
   - Выполняет пошаговую генерацию: сетка -> биомы -> POI -> границы -> препятствия.
3. **`MapGeneratorWindow` (OdinEditorWindow)**
   - UI редактора (`SteelSurge -> Level Editor`).
   - Управляет размерами карты, ориентацией (Horizontal/Vertical), сидом.
   - Отвечает за 2D-превью (генерация `Texture2D`) и финальную сборку сцены (создание `.unity` файла, настройка камеры и освещения).
4. **`HexGridService`**
   - Математика гексагональной сетки (осевые координаты `q, r`). Расчет расстояний, конвертация в мировые координаты, вычисление симметрии.

## Пайплайн генерации (`MapGenerator.cs`)

### 1. GenerateBaseGrid()
Создает плоскую сетку из базовых префабов гексов (`HexGrassPrefab`) заданного размера (`Width` x `Height`).

### 2. GenerateBiomes()
Использует `Mathf.PerlinNoise` со смещением (offset от seed) и `NoiseScale`.
Проходит по слоям биомов (`BiomeLayers`), отсортированным по `Threshold` (по убыванию). Если значение шума выше порога, гексу назначается соответствующий материал.

### 3. GeneratePoiSpots()
Определяет центры для баз (Keeps).
- По умолчанию: `(2, Height/2)` и симметричная точка на другой стороне.
- Очищает зону радиусом `PoiSpotRadius`, закрашивая ее специальным материалом (`PoiSpotMaterial`).

### 4. GenerateBorders() (Jagged Borders)
Создает горы по краям карты.
- Внешний контур (расстояние до края = 0) всегда горы.
- Вглубь до `MaxBorderDepth` горы спавнятся на основе `BorderNoiseScale` и `BorderNoiseThreshold`. Чем дальше от края, тем выше порог шума, что создает эффект "рваных" краев и естественных бухт.

### 5. GenerateObstacles()
Спавн деревьев и камней.
- **Защитные зоны:** Ничего не спавнится в радиусе `PoiSpotRadius` и `SafeZoneRadius` от баз, а также в центре карты (радиус 25% от ширины).
- **Кластеризация лесов (Warcraft 3 style):** 
  - Выбираются случайные центры кластеров.
  - Вокруг центров деревья спавнятся с высокой плотностью (`TreeClusterDensity`), образуя густые непроходимые леса.
  - На одном гексе может спавниться несколько деревьев (`TreesPerHex`, `TreeSpreadRadius`) со случайным вращением и смещением.
- **Камни:** Спавнятся хаотично вне лесов с вероятностью `RockDensity`.

## Особенности Editor Window

- **Ориентация (MapOrientation):** 
  - `Horizontal`: Ширина > Высота. Камера смотрит вдоль оси X.
  - `Vertical`: Высота > Ширина. Камера поворачивается на 90 градусов и смотрит вдоль оси Z.
- **Авто-масштабирование камеры:** `OrthographicSize` (как для обычной камеры, так и для Cinemachine) вычисляется динамически на основе высоты карты, чтобы арена всегда помещалась в кадр.
- **2D Preview:** Быстрая отрисовка карты в `Texture2D` попиксельно. Позволяет мгновенно перебирать сиды без инстанцирования тысяч GameObject'ов.
